Examples

Here are a few examples of XML reports generated by Classycle's Analyser. The values in the table show the size of the reports. Use the already rendered HTML versions if your Web browser isn't XSL enabled. Note, that the built-in XSL processor of your Web browser may needs a while for rendering large XML reports.

For the reports inner classes are merged with their outer classes except for Classycle itself.
